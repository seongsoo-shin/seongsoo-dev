## About this site

A hand-built, bilingual (English / Korean) static site — no framework, no build step.

- Plain HTML / CSS / JS, deployed on Vercel
- Language toggle via `data-i18n-en` / `data-i18n-ko` + `html[data-lang]`
- Dark editorial design system, custom typography, lightbox galleries
- SEO / Open Graph / JSON-LD, `robots.txt`, `sitemap.xml`, web manifest
- Print-friendly résumé via `@media print`
