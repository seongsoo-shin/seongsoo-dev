Neurodio

Neurodio

Audio-therapy mobile app · 3 locales
오디오 테라피 모바일 앱 · 3개국 언어

Overview

개요

An audio-therapy mobile app for NeuroGix with sleep, focus and meditation modes.

Built the mobile client in React Native and connected it to a Laravel + Vue admin.

NeuroGix의 오디오 테라피 모바일 앱입니다.

수면 · 집중 · 명상 모드로 동작하며, React Native 모바일 앱을 직접 작성해 Laravel + Vue 어드민과 연동했습니다.

What I built

담당 · 구현

  • React Native mobile client for iOS and Android.
  • Three-locale delivery: ko, en, ja.
  • Laravel + Vue content back office for sleep, focus and meditation modes.
  • iOS · Android 양면을 위한 React Native 모바일 클라이언트.
  • 한 · 영 · 일 3개 언어 빌드.
  • 수면 · 집중 · 명상 모드를 관리하는 Laravel + Vue 콘텐츠 백오피스.
