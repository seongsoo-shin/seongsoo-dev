GEM Platform

GEM Platform

IoT 3D-printing platform
IoT 3D 프린팅 플랫폼

Overview

개요

An IoT 3D-printing platform for (주)지이엠플롯폼 near Chunhae Health College.

Built the operations back office, the student e-learning views and a Node.js server that talks to networked 3D printers for model orders, remote-print queues and downloads.

춘해보건대학교 인근 (주)지이엠플롯폼의 IoT 3D 프린팅 플랫폼입니다.

운영 백오피스, 학생 e-러닝, 그리고 3D 프린터와 통신하는 Node.js 서버로 모델 주문 · 원격 출력 · 다운로드를 처리합니다.

What I built

담당 · 구현

  • Node.js server in IoT communication with networked 3D printers.
  • Model order flow, remote-print queue, model download.
  • Student e-learning views.
  • Operations back office for printer fleet and order management.
  • 네트워크 3D 프린터와 통신하는 Node.js 서버.
  • 모델 주문 흐름, 원격 출력 큐, 모델 다운로드.
  • 학생 e-러닝 화면.
  • 프린터 운영과 주문 관리를 묶은 운영 백오피스.
