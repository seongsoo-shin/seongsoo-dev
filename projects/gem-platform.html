GEM Platform

GEM Platform

3D model marketplace · remote-print platform
3D 모델 마켓플레이스 · 원격 출력 플랫폼

Overview

개요

A 3D-printing content platform for GEM Platform, separated from the student e-learning site.

I built the service flow for browsing printable models, requesting remote output and managing operational/admin views around the connected 3D-printer workflow.

GEM Platform의 3D 프린팅 콘텐츠·원격 출력 플랫폼입니다. 학생용 이러닝 사이트와 분리해 정리했습니다.

출력 가능한 모델 탐색, 원격 출력 요청, 연결된 3D 프린터 운영을 위한 서비스·관리자 흐름을 구현했습니다.

What I built

담당 · 구현

  • Model catalog and detail pages for printable 3D assets.
  • Remote-print request and download/order flow.
  • Operations/admin screens for managing course/order data.
  • Laravel + Node.js backend for service and printer-adjacent workflows.
  • 출력 가능한 3D 모델 카탈로그와 상세 화면.
  • 원격 출력 요청과 다운로드/주문 흐름.
  • 강의·주문 데이터를 관리하는 운영/관리자 화면.
  • 서비스와 프린터 연동 흐름을 위한 Laravel + Node.js 백엔드.
