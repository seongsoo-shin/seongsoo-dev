Project / 02

Le Soleil / Planterior

Mobile landing page design for a plant-interior service
식물 인테리어 서비스 모바일 랜딩 페이지 디자인

Overview

개요

A mobile landing page for Le Soleil / Planterior, a plant-interior service that turns commercial spaces into warmer, greener environments.

The page is structured as a long-form conversion flow: hero, problem framing, service value, client trust, reviews, portfolio examples and a closing CTA. The design keeps the tone soft and premium, using green accents, airy spacing, rounded cards and photo-led proof.

르솔레이유 플랜테리어는 상업 공간을 더 따뜻하고 자연스러운 분위기로 바꾸는 식물 인테리어 서비스의 모바일 랜딩 페이지입니다.

히어로, 문제 제기, 서비스 가치, 고객사 신뢰 요소, 리뷰, 포트폴리오 사례, 하단 CTA까지 이어지는 롱폼 전환 흐름으로 구성했습니다. 그린 포인트 컬러, 여백 중심 레이아웃, 둥근 카드, 사진 기반 증거를 사용해 부드럽고 고급스러운 인상을 만들었습니다.

What I built

담당 · 구현

  • Mobile-first long-form landing page layout in Figma.
  • Conversion flow from hero and problem framing to proof, portfolio and CTA.
  • Visual system with a planterior green palette, soft cards, light backgrounds and photo-led trust blocks.
  • Portfolio sections that show real-space examples and make the service easier to evaluate on mobile.
  • Figma 기반 모바일 우선 롱폼 랜딩 페이지 레이아웃.
  • 히어로와 문제 제기에서 신뢰 요소, 포트폴리오, CTA로 이어지는 전환 흐름.
  • 플랜테리어 그린 팔레트, 부드러운 카드, 밝은 배경, 사진 중심 신뢰 섹션으로 구성한 비주얼 시스템.
  • 실제 공간 사례를 보여주고 모바일에서 서비스를 쉽게 판단할 수 있게 하는 포트폴리오 섹션.
