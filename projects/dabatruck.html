Dabatruck

다바트럭

Top-truck advertising-matching platform
탑차 광고 매칭 플랫폼

Overview

개요

A top-truck advertising-matching platform connecting advertisers with truck operators.

Implemented real-time GPS route tracking, map-API matching, advertiser stats and operational dashboards across the full stack.

광고주와 탑차 운전자를 연결하는 광고 매칭 플랫폼입니다.

실시간 차량 이동경로 추적, 지도 API 매칭, 광고주 통계, 운영 대시보드까지 풀스택으로 구현했습니다.

What I built

담당 · 구현

  • Advertiser landing and matching flow.
  • Real-time vehicle GPS route tracking.
  • Map-API based advertiser ↔ driver matching.
  • Advertiser statistics dashboard.
  • Operations back office for accounts, matching and dispatch oversight.
  • 광고주 랜딩 페이지와 매칭 흐름.
  • 실시간 차량 GPS 이동경로 추적.
  • 지도 API 기반 광고주 ↔ 운전자 매칭.
  • 광고주 통계 대시보드.
  • 계정, 매칭, 배차 관리를 묶은 운영 백오피스.
