Dinx

Dinx

Anti-doping medication index for athletes (Japanese)
운동선수용 도핑 금지약물 인덱스 앱 (일본어)

Overview

개요

DINX (ドーピングインデックス) is a Japanese mobile app that lets athletes check whether a medication, supplement or prescription is banned under anti-doping rules before competition.

Delivered as an outsourced build for GY Seoul, with React Native on the front and a Laravel backend. Search runs by drug name, barcode, or a prescription QR code, and results return a clear banned / allowed verdict aligned to the WADA Prohibited List.

DINX (ドーピングインデックス) 는 운동선수가 대회 전에 의약품·서플먼트·처방약이 도핑 금지 대상인지 확인하는 일본어 모바일 앱입니다.

지와이서울 도급으로 진행했고, 프론트는 React Native, 백엔드는 Laravel로 구현했습니다. 의약품명·바코드·처방전 QR로 검색하면 WADA 금지목록 기준에 맞춰 금지 / 허용 여부를 명확히 판정해 보여줍니다.

What I built

담당 · 구현

  • Drug-name and manufacturer search against a medication index, returning a banned / allowed doping verdict.
  • Barcode scan and prescription QR lookup for fast point-of-purchase checks.
  • News feed of doping-violation cases and Prohibited-List updates.
  • Sport-by-sport FAQ, inquiry form, certificate request, and search history.
  • Full Japanese-language UI; React Native front end with a Laravel backend and admin.
  • 의약품명·제조사 검색으로 도핑 금지 / 허용 여부를 판정하는 의약품 인덱스.
  • 바코드 스캔과 처방전 QR 조회로 구매 현장에서 빠른 확인.
  • 도핑 위반 사례·금지목록 갱신을 알리는 신착정보 피드.
  • 스포츠 종목별 FAQ, 문의 폼, 증명서 발급 요청, 검색 이력.
  • 전체 일본어 UI · React Native 프론트 + Laravel 백엔드·관리자.
